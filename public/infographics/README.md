# Infographic Files

Place your infographic images (.png, .jpg, .webp) in this directory.

## Setup

The `InfographicLens` component provides:
- **Preview State**: Shows a cropped/teaser view with "VIEW SCHEMATIC" button
- **Lightbox Modal**: Full-screen view on click with smooth animations

## Current Infographics

- **mentalism-info.png** - Law 1: Mentalism infographic

## Adding More Infographics

To add infographics for other laws, update `lib/lawsData.js` and add the image files here:

- `mentalism-info.png`
- `correspondence-info.png`
- `vibration-info.png`
- `polarity-info.png`
- `rhythm-info.png`
- `cause-effect-info.png`
- `gender-info.png`

## Recommended Format

- **File Type**: PNG (for best quality with transparency)
- **Dimensions**: 1920x1080 or larger
- **Design**: Should work well when cropped at the top for preview

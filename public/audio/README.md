# Audio Files for Podcast Player

Place your .m4a podcast audio files in this directory.

## Current Setup

- **mentalism-pod.m4a** - Law 1: Mentalism podcast

## Adding More Podcasts

To add podcasts for other laws, update `lib/lawsData.js` and add the `.m4a` files here with these naming conventions:

- `mentalism-pod.m4a`
- `correspondence-pod.m4a`
- `vibration-pod.m4a`
- `polarity-pod.m4a`
- `rhythm-pod.m4a`
- `cause-effect-pod.m4a`
- `gender-pod.m4a`

## Format

- **File Type**: .m4a (MPEG-4 Audio)
- **Codec**: AAC preferred
- **Player**: "DEEPER DIVE PODCAST" custom player with black & gold design

## Specs

- **Format:** PNG or JPG
- **Dimensions:** 1920x1080 (16:9 aspect ratio recommended)
- **Size:** Optimized for web (~200-500 KB each)

## Usage

Thumbnails are displayed in the `LazyVideoPlayer` component before the video loads, providing a visual preview of the video content.
